# Predictive Maintenance - Turbofan Engines (DL Project)

Deep learning project for Remaining Useful Life (RUL) prediction using the NASA C-MAPSS turbofan degradation datasets.

## Project Structure

- `data/CMAPSSData/`: raw C-MAPSS files (kept local, not pushed)
- `data/outputs/`: processed datasets and metadata generated by preprocessing notebooks
- `notebooks/data/`: preprocessing notebooks (`01A` to `01D`)
- `notebooks/modeling/`: model development and experiments

## Quick Start

1. Create and activate a Python virtual environment.
2. Install required packages used in the notebooks.
3. Place C-MAPSS raw files in `data/CMAPSSData/`.
4. Run preprocessing notebooks in `notebooks/data/`.
5. Use `notebooks/modeling/modeling.ipynb` for training/evaluation.

## Git Notes

This repository ignores:

- local environment and editor files
- notebook checkpoints
- generated outputs in `data/outputs/`
- raw C-MAPSS data files (`train_*.txt`, `test_*.txt`, `RUL_*.txt`)

If you need to share outputs or data references, commit metadata/config files instead of raw/generated artifacts.
